## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, box-shadow offset |
| sm | 8px | Compact element spacing, tag badge gap, compare checkbox inset |
| md | 16px | Default element spacing, card content padding, collection card gap |
| lg | 24px | Section padding, collection header padding, comparison table cell padding |
| xl | 32px | Layout gaps, collection grid gap, compare bar inner padding |
| 2xl | 48px | Major section breaks, related scholarships top margin, collection browse page top spacing |
| 3xl | 64px | Page-level spacing, collection detail hero bottom margin |

Exceptions:
- Compare checkbox overlay: 8px inset from card top-right corner
- Floating compare bar: 16px horizontal padding, 12px vertical padding (12px is 3 * 4, compliant)
- Collection emoji display: 48px font-size (decorative, not spacing)

Source: Existing card.tsx uses `py-6` (24px) and `px-6` (24px), `gap-6` (24px). Scale matches established patterns.

## Typography

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Body | 14px (text-sm) | 400 (font-base) | 1.5 | Inter |

Notes:

- All headings use `font-heading` class (Archivo Black). Body and labels use `font-base` class (Inter).
- Collection card title: text-xl font-heading (20px Archivo Black)
- Collection detail page name: text-2xl font-heading (28px Archivo Black)
- Comparison table header row: text-sm font-heading (14px Archivo Black)
- Tag badge text: text-xs font-base (12px Inter)
- "Similar Scholarships" section heading: text-xl font-heading (20px Archivo Black)
- Compare bar scholarship names: text-sm font-base (14px Inter)

Source: Existing badge.tsx uses `text-xs font-base`, card.tsx uses `font-heading` for titles. Both weight 400.

## Interaction Contracts

### Compare Checkbox

| State | Visual | Behavior |
|-------|--------|----------|
| Default (desktop) | Hidden | Appears on card hover |
| Default (mobile) | Visible, unchecked | Always shown, top-right of card |
| Checked | Filled accent checkbox with checkmark | Card gets subtle accent border glow |
| Disabled (3 selected, this not selected) | Greyed out, no hover | Tooltip: "Maximum 3 scholarships" |
| Detail page variant | Full "Add to Compare" button | Below Apply button in hero section |

Source: D-54 (hover on desktop, always visible on mobile), D-99 (keyboard: Tab + Space/Enter)

### Floating Compare Bar

| State | Visual | Behavior |
|-------|--------|----------|
| 0 selected | Hidden | No bar rendered |
| 1 selected | Bar visible, 1 scholarship chip, "Compare" button disabled | Chip shows title (truncated 20ch) + X remove |
| 2 selected | Bar visible, 2 chips, "Compare" button enabled | Click "Compare" navigates to /scholarships/compare?s=slug1,slug2 |
| 3 selected | Bar visible, 3 chips, "Compare" button enabled, "Clear all" link | Max reached, no more can be added |

Position: Fixed bottom, 16px from edges, z-50. Mobile: full-width with 8px horizontal margin. Desktop: centered, max-width 640px.
Animation: slide-up from bottom on appear, slide-down on dismiss. Respects `prefers-reduced-motion` (instant show/hide).
ARIA: `aria-live="polite"` announces selection changes to screen readers (D-100).

Source: D-54, D-56, D-66, D-110 (fixed bottom mobile), D-100 (ARIA), D-102 (reduced motion)

### Comparison Table

| Feature | Specification |
|---------|--------------|
| Layout | Semantic `<table>` with `<th scope="col">` for scholarship names and `<th scope="row">` for field labels |
| Fields (rows) | Provider, Host Country (with flag), Degree Levels, Funding Type, Award Amount, Application Deadline (with urgency badge), Eligible Nationalities (count + top 3), Fields of Study, Prestige Tier (badge), Tags (first 3 + count) |
| Difference highlighting | Cells where values differ across columns get `bg-[var(--compare-diff-bg)]` background |
| Column header | Scholarship title (linked to detail page), prestige badge, "Apply" button (links to application_url) |
| Mobile | Horizontal scroll with sticky first column (field names). Minimum column width: 200px. |
| Empty column | If comparing 1-2, rightmost column shows "Add scholarship" SearchToAdd component |
| Remove | X button in column header to remove scholarship from comparison |

Source: D-57 (difference highlighting), D-58 (Apply + detail link), D-59 (search-to-add), D-64 (mobile horizontal scroll + sticky first column), D-98 (semantic table, th scope, ARIA)
