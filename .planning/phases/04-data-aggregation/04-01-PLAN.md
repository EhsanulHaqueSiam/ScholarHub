    ```typescript
    };
    ```

    **getTrustRank(category: string): number**:
    - Return `TRUST_RANK[category] ?? 0`

    **resolveField\<T\>(candidates: Array\<{ value: T | undefined | null; category: string; scrapedAt: number }\>): T | undefined** — per D-05/D-06:
    - Filter out null/undefined/"" values

    - Return first element's value, or undefined if none

    - Try regex `/\b(20\d{2})\b/` on title first
    - If no match and deadline exists, return `new Date(deadline).getFullYear()`
    - Otherwise return null

    **shouldArchive(deadline: number | undefined | null): boolean** — per D-11:
    - If no deadline, return false
    - Return `Date.now() - deadline > 30 * 24 * 60 * 60 * 1000`

    **computeExpectedReopenMonth(historicalDeadlines: number[]): number | null** — per D-10:
    - If empty array, return null
    - Map each deadline to its month (1-12) via `new Date(d).getMonth() + 1`
    - Count frequency of each month
    - Return the most common month (first in case of tie)

    **toSlug(title: string): string** — reuse pattern from seed.ts:
    - `title.toLowerCase().replace(/[^a-z0-9]+/g, "-").replace(/^-|-$/g, "")`
    - Truncate to 80 chars
