  <behavior>
    Weight distribution per D-17, D-18:
    - nationality: 35 points (heaviest per D-18)
    - degree: 25 points
    - fieldOfStudy: 20 points
    - fundingType: 10 points
    - demographics: 10 points
    - Total: 100 points

    Tier thresholds per D-17:
    - Strong: score >= 80
    - Good: score >= 50 and < 80
    - Partial: score >= 20 and < 50
    - Possible: score < 20 OR scholarship has too many unknown dimensions

    Scoring rules:
    - Nationality (35 pts): Full points if scholarship has no nationality restrictions OR user's nationality is in the eligibility list. Zero if user's nationality is not in a restricted list. Unknown if scholarship has no nationality data.
    - Degree (25 pts): Full points if user's degree level is in scholarship's degree_levels array. Zero if not. Unknown if scholarship has no degree data.
    - Field of study (20 pts): Full 20 pts if any user field matches scholarship fields. Partial 10 pts if scholarship has no field restriction (open to all). Zero if no overlap. Unknown if no field data on scholarship.
    - Funding type (10 pts): Full points if scholarship funding_type matches user preference. Partial 5 pts if no preference set. Zero if mismatch. Unknown if no funding data.
    - Demographics/gender (10 pts): Full points if scholarship has no gender requirement OR matches user gender. Zero if scholarship requires a specific gender the user is not. Unknown if user set "prefer_not_to_say" and scholarship has gender requirement.

    Possible match rule per D-19: If 3+ dimensions have "unknown" status (scholarship missing data), classify as "possible" regardless of score.

    Deadline rules per D-21:
    - isClosingSoon(deadline): true if deadline exists and is within 30 days from now
    - Expired scholarships (deadline < now) should be excluded before scoring

    Test cases (RED phase):
    1. Perfect match (all dimensions match) -> Strong tier, score ~100
    2. Nationality + degree match, no field data -> Good tier
    3. Only nationality matches, everything else unknown -> Possible tier (3+ unknowns)
    4. Nationality mismatch on restricted scholarship -> score includes 0 for nationality
    5. Open-to-all nationality scholarship -> full nationality points
    6. Field partial match (scholarship has no restrictions) -> 10 pts partial
    7. isClosingSoon returns true for deadline 15 days away
    8. isClosingSoon returns false for deadline 45 days away
    9. scoreAllScholarships groups results by tier correctly
  </behavior>

    ```typescript
    import type { StudentProfile, EligibilitySummary, ScoredScholarship, MatchTier, MatchBreakdown, MatchStatus } from "./types";

    export interface ScoringWeights {
      nationality: number;
      degree: number;
      fieldOfStudy: number;
      fundingType: number;
      demographics: number;
    }

    export const DEFAULT_WEIGHTS: ScoringWeights = {
      nationality: 35,
      degree: 25,
      fieldOfStudy: 20,
      fundingType: 10,
      demographics: 10,
    };

    export function scoreScholarship(
      scholarship: EligibilitySummary,
      profile: StudentProfile,
      weights: ScoringWeights = DEFAULT_WEIGHTS,
    ): ScoredScholarship { ... }

    export function scoreAllScholarships(
      scholarships: EligibilitySummary[],
      profile: StudentProfile,
    ): Record<MatchTier, ScoredScholarship[]> { ... }

    export function isClosingSoon(deadline: number | undefined): boolean {
      if (!deadline) return false;
      const thirtyDays = 30 * 24 * 60 * 60 * 1000;
      return deadline > Date.now() && deadline - Date.now() < thirtyDays;
    }
    ```

    The scoreScholarship function:
    1. Score each dimension independently, tracking earned points and possible points
    2. Count unknowns -- if 3+ unknowns, force tier to "possible" per D-19
    3. Calculate percentage = (earned / possible) * 100
    4. Map to tier: >= 80 Strong, >= 50 Good, >= 20 Partial, else Possible
    5. Return { scholarship, tier, score, breakdown }

    The scoreAllScholarships function:
    1. Map each scholarship through scoreScholarship
    2. Group results into Record<MatchTier, ScoredScholarship[]>
    3. Sort within each tier by score descending
