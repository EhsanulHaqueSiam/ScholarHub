## Typography

| Role | Font | Size | Weight | Line Height |
|------|------|------|--------|-------------|
| Body | Inter | 16px | 400 (regular) | 1.5 |
| Label | Inter | 14px | 600 (semibold) | 1.4 |
| Heading | Archivo Black | 20px | 400 | 1.2 |
| Display | Archivo Black | 48px (desktop) / 32px (mobile) | 400 | 1.1 |

Notes:
- Archivo Black only ships weight 400 -- it is a display-weight face by design. Do not apply `font-bold` to Archivo Black; it is already bold.
- Inter is used for all non-heading text. Use weight 400 for body, weight 600 for labels, badges, and button text.
- Tailwind v4 font configuration via `@theme` in CSS:

```css
@theme inline {
  --font-heading: "Archivo Black", ui-sans-serif, system-ui, sans-serif;
  --font-base: "Inter", ui-sans-serif, system-ui, sans-serif;
  --font-weight-heading: 400;
  --font-weight-base: 400;
}
```

Google Fonts import (add to `index.html` `<head>`):

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Archivo+Black&family=Inter:wght@400;600&display=swap" rel="stylesheet">
```

## Color

All colors use oklch values for Tailwind v4 compatibility. Hex approximations shown for reference only.

### Light Mode (default)

| Role | Value | Hex Approx | Usage |
|------|-------|------------|-------|
| Dominant (60%) -- background | `oklch(93.46% 0.0304 254.32)` | ~#e3dff2 | Page background |
| Secondary (30%) -- surfaces | `oklch(100% 0 0)` | #ffffff | Cards, content containers |
| Accent (10%) -- main | `oklch(67.47% 0.1725 259.61)` | ~#6366f1 | See reserved-for list below |
| Foreground -- text | `oklch(0% 0 0)` | #000000 | All body text, headings |
| Main foreground -- text on accent | `oklch(0% 0 0)` | #000000 | Text rendered on accent-colored backgrounds |
| Border | `oklch(0% 0 0)` | #000000 | All borders (2px solid) |
| Overlay | `oklch(0% 0 0 / 0.8)` | rgba(0,0,0,0.8) | Modal/dialog backdrop |

### Dark Mode

| Role | Value | Hex Approx |
|------|-------|------------|
| Background | `oklch(29.12% 0.0633 270.86)` | ~#1a1643 |
| Secondary background | `oklch(23.93% 0 0)` | ~#3b3b3b |
| Foreground | `oklch(92.49% 0 0)` | ~#ebebeb |
| Main foreground | `oklch(0% 0 0)` | #000000 |
| Border | `oklch(0% 0 0)` | #000000 |
| Ring (focus) | `oklch(100% 0 0)` | #ffffff |

### Accent Reserved For

The accent color (`--main`) is used ONLY on these elements:

1. Primary action buttons (default variant)
2. Badge backgrounds (default variant)
3. Active/selected states on interactive elements
4. Focus ring fills (not outlines -- outlines use `--border`)

Accent is NOT used for: links (use underlined black text), headings (use foreground), borders (always black), backgrounds (use `--background` or `--secondary-background`).

### Neo-Brutalism Visual Properties

| Property | Value | CSS Variable |
|----------|-------|--------------|
| Border width | 2px solid | hardcoded in component classes |
| Border color | `--border` (black) | `var(--border)` |
| Border radius | 5px | `--border-radius: 5px` / `rounded-base` |
| Box shadow X | 4px | `--box-shadow-x: 4px` |
| Box shadow Y | 4px | `--box-shadow-y: 4px` |
| Shadow value | `4px 4px 0px 0px var(--border)` | `--shadow` / `shadow-shadow` |
| Hover behavior | Translate to consume shadow | `hover:translate-x-boxShadowX hover:translate-y-boxShadowY hover:shadow-none` |

No blur on shadows. No gradients. No rounded corners beyond 5px. These are defining characteristics of the neo-brutalism style.

### CSS Custom Properties Block

Place in `web/src/index.css` after `@import "tailwindcss"`:

```css
@import "tailwindcss";

@custom-variant dark (&:is(.dark *));

:root {
  --border-radius: 5px;
  --box-shadow-x: 4px;
  --box-shadow-y: 4px;
  --reverse-box-shadow-x: -4px;
  --reverse-box-shadow-y: -4px;

  --heading-font-weight: 400;
  --base-font-weight: 400;

  --background: oklch(93.46% 0.0304 254.32);
  --secondary-background: oklch(100% 0 0);
  --foreground: oklch(0% 0 0);
  --main-foreground: oklch(0% 0 0);

  --main: oklch(67.47% 0.1725 259.61);
  --border: oklch(0% 0 0);
  --ring: oklch(0% 0 0);
  --overlay: oklch(0% 0 0 / 0.8);

  --shadow: var(--box-shadow-x) var(--box-shadow-y) 0px 0px var(--border);
}

.dark {
  --background: oklch(29.12% 0.0633 270.86);
  --secondary-background: oklch(23.93% 0 0);
  --foreground: oklch(92.49% 0 0);
  --main-foreground: oklch(0% 0 0);

  --border: oklch(0% 0 0);
  --ring: oklch(100% 0 0);

  --shadow: var(--box-shadow-x) var(--box-shadow-y) 0px 0px var(--border);
}

@theme inline {
  --font-heading: "Archivo Black", ui-sans-serif, system-ui, sans-serif;
  --font-base: "Inter", ui-sans-serif, system-ui, sans-serif;
  --font-weight-heading: var(--heading-font-weight);
  --font-weight-base: var(--base-font-weight);

  --color-main: var(--main);
  --color-background: var(--background);
  --color-secondary-background: var(--secondary-background);
  --color-foreground: var(--foreground);
  --color-main-foreground: var(--main-foreground);
  --color-border: var(--border);
  --color-overlay: var(--overlay);
  --color-ring: var(--ring);

  --spacing-boxShadowX: var(--box-shadow-x);
  --spacing-boxShadowY: var(--box-shadow-y);
  --spacing-reverseBoxShadowX: var(--reverse-box-shadow-x);
  --spacing-reverseBoxShadowY: var(--reverse-box-shadow-y);

  --radius-base: var(--border-radius);

  --shadow-shadow: var(--shadow);
}

@layer base {
  body {
    @apply bg-background text-foreground font-base;
  }

  h1, h2, h3, h4, h5, h6 {
    @apply font-heading;
  }
}
```

### Placeholder Page Layout

The placeholder page renders a single centered card on the lavender background:

```
+--------------------------------------------------+
|                                                  |
|              [Coming Soon] (badge)               |
|                                                  |
|              ScholarHub                           |
|              (display heading, Archivo Black)     |
|                                                  |
|              Find your scholarship.               |
|              Fund your future.                    |
|              (heading, Archivo Black 20px)         |
|                                                  |
|              We are building the most             |
|              comprehensive international          |
|              scholarship directory. Every         |
|              scholarship. Every country.          |
|              One place.                           |
|              (body, Inter 16px)                    |
|                                                  |
+--------------------------------------------------+
```

- Page is vertically and horizontally centered (`min-h-screen flex items-center justify-center`)
- Card uses `bg-secondary-background` (white), `border-2 border-border`, `shadow-shadow`
- Badge uses default variant (accent background, black text)
- Max width of card container: 480px
- Padding inside card: 32px (xl)
- Text alignment: center
- Mobile: card fills width with 16px horizontal margin
