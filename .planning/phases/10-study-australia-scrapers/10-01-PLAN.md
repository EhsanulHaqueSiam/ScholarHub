    **Step 2: Create the InertiaScraper** at `scraping/src/scholarhub_pipeline/scrapers/inertia_scraper.py`:

    The class must:
    - Extend `BaseScraper`
    - Import: `re`, `asyncio`, `httpx`, `structlog`, `BaseScraper`, `get_random_ua`
    - Define a custom exception class `InertiaVersionMismatchError(Exception)` with a `location` attribute
    - Define a module-level helper `map_study_australia_record(item: dict) -> dict` that:
      - Extracts `organisations[0].name` as `provider_organization` (default empty string if no orgs)
      - Extracts `level_of_studies[].name` as list into `degree_level`
      - Extracts `field_of_studies[].name` as list into `field_of_study`
      - Maps: name->title, description->description, eligibility->eligibility_criteria, closing_date->application_deadline, str(amount_annual)->award_amount (None if falsy), amount_comment->funding_details, web_address->application_url, scholarship_country_name->host_country
      - Builds source_url as `https://search.studyaustralia.gov.au/scholarship/{slug}/{id}`
      - Includes `is_for_international_students` boolean
    - Define `_extract_inertia_version(html: str) -> str | None` as a `@staticmethod`:
      - Use regex `r'version&quot;:&quot;([a-f0-9]+)&quot;'` to find version hash
      - Return the captured group or None
    - Implement `async def scrape(self) -> list[dict]`:
      - Create httpx.AsyncClient with `User-Agent` from `get_random_ua()`, `timeout=30.0`, `follow_redirects=True`
      - **Initial HTML request**: GET `self.config.url` to fetch SSR HTML
      - Extract version via `_extract_inertia_version`; raise `ValueError("Could not extract Inertia version hash from {self.config.url}")` if None
      - Track `self.bytes_downloaded += len(initial.content)`
      - **Pagination loop**: page starts at 1, max_pages from `self.config.pagination.get("max_pages", 200)` if pagination exists, else 200
      - Each page: GET `{self.config.url}?page={page}` with headers:
        ```
        X-Inertia: true
        X-Inertia-Version: {version}
        X-Requested-With: XMLHttpRequest
        Accept: text/html, application/xhtml+xml
        ```
      - Wrap the Inertia GET in a try/except for `httpx.HTTPStatusError`. If `response.status_code == 409`:
        - Re-fetch the initial HTML page
        - Extract new version
        - Retry the same page URL with new version
        - If retry also fails, raise
        - Track max 1 retry per 409
      - `response.raise_for_status()`, `self.bytes_downloaded += len(response.content)`
      - Parse JSON: `data = response.json()`
      - Extract items_key from `self.config.selectors.get("items_key", "scholarships")`
      - Navigate: `items_data = data.get("props", {}).get(items_key, {})`
      - Get `items = items_data.get("data", [])` and `meta = items_data.get("meta", {})`
      - If not items: break
      - For each item: if `self.config.field_mappings` is non-empty, use `self.apply_field_mappings(item)` then `self.process_record(mapped)`. Otherwise use `map_study_australia_record(item)` then `self.process_record(mapped)`. Append to records, increment `self.records_found`.
      - Check pagination: if `page * meta.get("per_page", 10) >= meta.get("total", 0)`: break
      - Increment page, sleep `self.config.rate_limit_delay`
      - Return records
