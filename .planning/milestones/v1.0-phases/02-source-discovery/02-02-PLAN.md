    **Step 2: Create scraping/scripts/validate_sources.py**
    Structure:
    ```python
    #!/usr/bin/env python3
    """Validate source catalog URLs for reachability and detect duplicates."""

    import asyncio
    import json
    import sys
    from pathlib import Path
    from urllib.parse import urlparse, urlunparse

    import aiohttp

    CONCURRENCY = 20
    TIMEOUT = aiohttp.ClientTimeout(total=10)
    SKIP_FILES = {"schema.json", "validation_report.json"}
    ```

    Functions to implement:
    - `normalize_url(url: str) -> str` -- Force HTTPS, strip `www.`, strip trailing slash (keep `/` for root), strip query params and fragment. Use `urlparse`/`urlunparse`.
    - `load_all_sources(source_dir: Path) -> list[dict]` -- Glob `*.json` in source_dir, skip files in SKIP_FILES, parse each as JSON array, return flat list of all entries with a `_file` key added for reporting.
    - `find_duplicates(sources: list[dict]) -> list[dict]` -- Normalize all URLs, group by normalized URL, return list of dicts `{"normalized_url": str, "entries": [{"name": str, "url": str, "file": str}]}` for any normalized URL with 2+ entries.
    - `async check_url(session: aiohttp.ClientSession, semaphore: asyncio.Semaphore, entry: dict) -> dict` -- Async URL check with semaphore. Return `{"name": str, "url": str, "status": int|None, "final_url": str|None, "redirected": bool, "reachable": bool, "error": str|None}`. Follow redirects. 10s timeout.
    - `async validate_all(sources: list[dict]) -> list[dict]` -- Create semaphore(CONCURRENCY), create single `aiohttp.ClientSession`, gather all check_url tasks.
    - `write_report(results: list[dict], duplicates: list[dict], output_path: Path) -> None` -- Write JSON report with `{"checked_at": ISO timestamp, "total": int, "reachable": int, "unreachable": int, "redirected": int, "duplicates": [...], "results": [...]}`.
    - `def main() -> None` -- Parse optional `--source-dir` arg (default `scraping/sources`), load sources, find duplicates, run async validation, write `validation_report.json` to source_dir, print summary to stdout.

    Print summary format:
    ```
    Source Catalog Validation Report
    ================================
    Total sources: NN
    Reachable: NN
    Unreachable: NN
    Redirected: NN
    Duplicate URLs: NN

    [If unreachable:]
    Unreachable URLs:
      - SourceName: https://... (error: ...)

    [If duplicates:]
    Duplicate URLs (after normalization):
      - https://normalized... found in: file1.json (Name1), file2.json (Name2)
    ```

    Use `if __name__ == "__main__": main()` entry point.
