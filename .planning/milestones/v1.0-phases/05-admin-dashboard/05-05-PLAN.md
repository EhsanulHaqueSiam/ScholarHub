The admin/index.tsx needs to manage EditPanel state:

```typescript

const [editingScholarship, setEditingScholarship] = useState<{
  id: Id<"scholarships">;
  title: string;
} | null>(null);
```

Pass the `setEditingScholarship` handler down to ReviewQueue -> QueueRow:
- ReviewQueue receives `onEditScholarship: (id, title) => void` prop

- QueueRow's "Edit" button calls `onEdit()`

```typescript
<EditPanel
  open={!!editingScholarship}

  scholarshipTitle={editingScholarship?.title ?? ""}
  onClose={() => setEditingScholarship(null)}
/>
```
