- Deadline line: Badge with urgency variant from `getDeadlineUrgency(applicationDeadline)` + urgency label from `urgencyLabelMap`. Next to it, formatted date using `new Intl.DateTimeFormat("en-US", { dateStyle: "long" }).format(new Date(applicationDeadline))`. Then client-only countdown from `useCountdown(applicationDeadline)` rendered as `({daysLeft} days left)` or `(1 day left)` or `(Due today)`. If deadline is expired (daysLeft <= 0), show "Applications Closed" text instead of countdown.
- Apply Now button: If applicationUrl exists and deadline not expired, render `<a href={applicationUrl} target="_blank" rel="noopener noreferrer">` styled as `<Button variant="default" size="lg" className="w-full md:w-auto" asChild>` with text "Apply Now" and `aria-label="Apply to {title} (opens in new tab)"`. If no applicationUrl, render disabled Button with text "Application Link Unavailable". If expired, render disabled Button with text "Applications Closed".
- Accept a `ref` prop via `React.forwardRef` so the parent can attach IntersectionObserver to the hero div.
