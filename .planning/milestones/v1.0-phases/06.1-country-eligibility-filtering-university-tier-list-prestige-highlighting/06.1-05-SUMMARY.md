## Decisions Made
- CountrySelector uses Radix Popover rather than Radix Select because Select doesn't support multi-select well -- Popover allows custom checkbox list with search input
- EligibilityFilterBar compact mode auto-activates when both `from` and `to` filters are set, with a manual "Edit" button to expand and "Collapse" to re-compact
- QuickFilters "Open Now" is derived state (active when neither closing_soon nor show_closed is true) rather than an explicit URL parameter, since it represents the default view
- SearchBar combines Convex searchSuggestions (scholarship title matches) with client-side FIELDS_OF_STUDY matching in the dropdown, showing category matches with "Filter by:" prefix
- FilterPanel mobile bottom sheet uses Radix Dialog with `fixed bottom-0 inset-x-0` positioning rather than a separate sheet library
